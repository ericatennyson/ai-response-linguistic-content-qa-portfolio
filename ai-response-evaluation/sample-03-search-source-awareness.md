# Sample 03: Search and Source Awareness

## Prompt

Can I still apply for this online AI training role this week?

## Response A

Yes, you can probably still apply. These roles are usually open for a long time.

## Response B

This depends on the current status of the job posting. A strong answer should check the official posting or application page, confirm whether the role is still open, and summarize any deadline, eligibility, or availability information listed there.

## Evaluation

**Preferred Response:** Response B

## Rationale

Response B is stronger because the user is asking about a current opportunity, and job availability can change quickly. It correctly identifies that the answer should be grounded in the official posting instead of relying on a general assumption.

Response A may sound reassuring, but it is not reliable. It could mislead the user if the role has closed, changed requirements, or stopped accepting applications.

## Quality Criteria

- Current information handling: Strong in Response B
- Search/source awareness: Strong in Response B
- Accuracy risk: Higher in Response A
- User usefulness: Strong in Response B
