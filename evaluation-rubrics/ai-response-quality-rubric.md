# AI Response Quality Rubric

I evaluate AI-generated responses based on the following criteria:

## 1. User Intent Alignment

Does the response answer what the user actually asked?

## 2. Helpfulness

Does it provide useful, practical, and complete support?

## 3. Accuracy

Does it avoid unsupported claims, outdated information, or misleading statements?

## 4. Personalization Quality

Does it use user context appropriately without over-inferring or forcing a connection?

## 5. Search/Source Awareness

When current, specific, or verifiable information is needed, does the response identify the need for reliable sources?

## 6. Safety and Sensitivity

Does it handle sensitive topics carefully and avoid harmful or overconfident advice?

## 7. Clarity

Is the response easy to understand, well-structured, and appropriately concise?

## 8. Rationale Quality

Can the evaluator clearly explain the decision using evidence from the prompt and responses?
